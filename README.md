## About

Gizmo Control is a Unity development tool that centralizes gizmo rendering logic through a simple interface. It allows developers to control gizmo visibility per script via the Inspector, with modes like Never, SelectedOnly, NotSelectedOnly, and Always â€” eliminating the need to change code to adjust gizmo display. This simplifies gizmo management, enhances debugging clarity, and improves overall development workflow.

### Features
- Simplified Gizmo Drawer with a unified interface
- Gizmo draw mode selectable in inspector
